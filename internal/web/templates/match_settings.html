Matchmaking Policy

Distance-weighted policy schedules all valid pairs, but assigns far-away Elo opponents fewer games. Scale controls how fast that decay happens.

Mirror matches use the same formula with Elo distance 0, so they are weighted like the closest possible opponent pair.

Queue refill balances underplayed pairs first. Non-mirror pairs are scheduled in both colors; mirror pairs are scheduled directly.
